# youtube-mp3-downloader
Allows you to download mp3 from youtube video

## Usage
1. Clone the project:
    ```
    git clone https://github.com/molotochok/youtube-mp3-downloader.git
    ```
1. Install packages:
    ```
    npm i
    ```
1. Download mp3 file:
    ```
    npm run download -- <youtube-video-id>
    ```
    Example: 
    ```
    npm run download -- nVjsGKrE6E8
    ```
